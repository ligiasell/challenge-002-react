# Styles folder

Global styles used by the app

Files starting with `_` indicates that this files shouldn't be used directly on components (they are imported by other global styles - e.g. settings.css)

### Folder structure

* styles

  * settings.css
  * \_colors.css
  * \_reset.css
  * \_grid.css
  * \_font-weights.css
  * ...
